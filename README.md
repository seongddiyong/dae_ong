# 데옹의 연습장
### 목표
1. 썼던 기술은 보강하고 정리
2. 새로운 기술도 적용해보고 정리
3. 테스트코드를 꼭 작성해보기

## Tools
- SpringBoot
- Java 17

## Rule
1. Issues에 task 등록
2. Issues에 맞는 branch 생성 후 진행 상황 표시
  - Branch : __ / __ / #IssueNum
  - etc : [ __ / __ / #IssueNum ] description
3. 각 파일의 readme 생성 후 설명 (main page에 링크)
4. 이상 나머지 나중에 추가
